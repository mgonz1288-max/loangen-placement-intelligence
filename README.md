# Placement Intelligence Prototype

A self-contained, clickable internal-tool prototype for a completed small-business financing application.

## Run

Open `index.html` in a browser, or serve the folder with any static file server.

For example:

```powershell
python -m http.server 4173
```

Then visit `http://localhost:4173`.

No build step, API keys, or dependencies are required. All business and recommendation data is fictional prototype data.
